# IVIS

2 different systems for searching, inserting, deleting key-value pair entires for use in in-vehicle information systems

One uses Hash maps and an array list (IVIS)
The other systems uses doubly linked lists(IVIS_system2)

To Run the Programs issue following commands 

For system 1:
javac IVIS
java IVIS

You will be prompted with a menu, follow the options and chose if you want to insert, delete or search. The menu will be prompted each time so the system is very straightforward to use 

For system 2:

javac IVIS_system2
java IVIS_system2

Similarly menu prompt as system 1s
